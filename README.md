# insta-clone

Instagram Clone with >> Express, Prisma, React, React Native

## Features

### User

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Log in)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow / Unfollow user
- [x] Edit my profile
- [x] See user profile
- [x] See my profile
- [x] See full photos
- [x] Upload a photo
- [x] Edit the Photo
- [x] See the feed
- [x] Send DM
- [x] See all rooms
- [x] See room
- [x] Receive DM
