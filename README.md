# insta-clone

Instagram Clone with >> Express, Prisma, React, React Native

## Features

### User

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Log in)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [ ] See the feed
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] See all photos
- [ ] Upload a photo
- [ ] Edit the Photo
