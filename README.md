# insta-clone

Instagram Clone with >> Express, Prisma, React, React Native

## Features

### User

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Log in)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow / Unfollow user
- [x] Edit my profile
- [ ] See my profile
- [x] See user profile
- [ ] See the feed
- [ ] See all photos
- [ ] Upload a photo
- [ ] Edit the Photo
